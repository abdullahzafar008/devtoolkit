Common Questions

How do I report a broken tool? +
You can fill out the contact form above, selecting "Bug Report" as the subject. Please describe the error, what input you entered, and any browser console logs you observed.
Can I suggest a new tool? +
Absolutely! Select "Tool Request" in the form above and describe the features of the utility you need. Our development team regularly adds new modules based on user votes.
Is DevToolKit really free? +
Yes, 100% free! Every tool is executed fully in your local browser sandbox, requiring no remote API subscriptions or fees. You can use them as much as you like.
Do you offer API access? +
Currently, DevToolKit is designed as a browser-first interactive platform. Because computations occur client-side, we do not offer external remote API endpoints.
How do I delete my account? +
Log in, click your username avatar dropdown, go to Profile settings, scroll to the bottom Danger Zone, and click the "Delete Account" button. All history records will be purged immediately.
