Frequently Asked Questions

Does Sentence Case capitalize proper nouns? +
Since proper nouns capitalization depends on dictionaries database, standard Sentence Case filters only prioritize character capitalization directly following sentence markers like '.', '?', or '!'.
How does Invert Case behave? +
Invert Case maps lowercase characters to uppercase, and uppercase characters to lowercase, leaving spaces, digits, and special glyphs unchanged.
Is there a limit on text size? +
No. The Javascript string buffer runs locally in your device RAM memory and handles complete essays or logs databases instantly.
