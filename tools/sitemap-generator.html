Frequently Asked Questions

Where do I upload the sitemap.xml file? +
Upload the file directly into the top-level root public folder of your site's server. For example: `https://yourdomain.com/sitemap.xml`.
What is the difference between an XML sitemap and HTML sitemap? +
XML sitemaps are formatted strictly in XML schema so search engine bots can easily digest them. HTML sitemaps are intended as regular web pages featuring navigational link hierarchies for actual human visitors.
Does a sitemap guarantee that search engines will index my pages? +
No. Sitemaps ensure that search engine crawlers can *find* all your pages. However, search engines decide whether to index pages based on quality, crawl budgets, uniqueness, and on-page authority metrics.
How do I submit my sitemap to search engines? +
Submit your sitemap through Google Search Console (under Sitemaps menu) and Bing Webmaster Tools. You should also add a directive link in your robots.txt file, e.g. `Sitemap: https://yourdomain.com/sitemap.xml`.
