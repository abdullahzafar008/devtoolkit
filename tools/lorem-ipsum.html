Frequently Asked Questions

Where did the standard standard passage come from? +
It originates from a passage written by Cicero in 45 BC. A type specialist scrambled the words in the 1500s to create a book of type samples.
How does list items mode behave? +
Selecting "List Items" structures each item as a bullet point. With HTML enabled, items are enveloped inside standard list item tags (e.g. <li>Lorem ipsum...</li>).
Is there a limit on text size? +
The tool caps paragraph generation at 100 items to keep browser performance optimal and avoid rendering slowdowns.
